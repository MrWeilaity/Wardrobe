### 5. 用户管理 👥

```
┌─────────────────────────────────────────────────────────────────┐
│  用户管理                               [📥 导出用户数据]       │
├─────────────────────────────────────────────────────────────────┤
│                                                                   │
│  🔍 搜索: [_______________]  [全部角色▼]  [按ID排序▼]          │
│                                                                   │
├───┬─────────┬────────────────┬─────────┬────────┬───┬───┬───┬──┤
│ID │ 用户名  │ 邮箱           │ 角色    │ 注册   │衣物│穿搭│行程│操作│
├───┼─────────┼────────────────┼─────────┼────────┼───┼───┼───┼──┤
│ 1 │ admin   │ admin@ex.com   │🔴管理员 │2024-01 │ 45│ 12│  3│👁️⬇️🗑️│
│ 2 │ zhangsan│ zhang@ex.com   │🔵用户   │2024-02 │ 23│  8│  2│👁️⬆️🗑️│
│ 3 │ lisi    │ li@example.com │🔵用户   │2024-03 │ 15│  5│  1│👁️⬆️🗑️│
└───┴─────────┴────────────────┴─────────┴────────┴───┴───┴───┴──┘
```

**功能详解**:

#### 🔍 搜索功能
- 实时搜索用户名
- 实时搜索邮箱
- 即时响应，无需点击

#### 🏷️ 筛选功能
- **全部角色**: 显示所有用户
- **仅管理员**: 只显示管理员
- **仅用户**: 只显示普通用户

#### 📊 排序功能
- **按ID**: 默认排序
- **按用户名**: 字母顺序
- **按注册时间**: 最新注册优先
- **按衣物数量**: 最活跃用户优先

#### 🎯 操作按钮
- 👁️ **查看**: 打开详情模态框
- ⬆️ **提升**: 授予管理员权限
- ⬇️ **降级**: 移除管理员权限
- 🗑️ **删除**: 删除用户及数据
