# 智衣橱 - 快速开始指南

## 🚀 运行系统（10分钟搞定）

### 第零步：准备 MySQL 数据库

**安装 MySQL 5.7 并创建数据库：**

```bash
# 登录 MySQL
mysql -u root -p

# 创建数据库
CREATE DATABASE wardrobe CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

或者直接运行 SQL 脚本：
```bash
mysql -u root -p < backend/src/main/resources/schema.sql
```

**配置数据库密码：**
如果您的 MySQL 密码不是 `root`，请修改 `backend/src/main/resources/application.yml` 中的密码配置。

详细配置说明请查看：`backend/数据库配置说明.md`

### 第一步：启动后端

打开终端，执行：

```bash
cd backend
mvn spring-boot:run
```

✅ 看到 "Started WardrobeApplication" 表示后端启动成功！
- 后端地址：http://localhost:8080
- 数据库：MySQL 5.7 (localhost:3306/wardrobe)

### 第二步：启动前端

打开新的终端，执行：

```bash
cd frontend
npm install
npm run dev
```

✅ 看到 "Local: http://localhost:5173" 表示前端启动成功！

### 第三步：使用系统

打开浏览器访问：**http://localhost:5173**

1. 点击"立即注册"创建账号
2. 登录后即可使用系统

## ❓ 常见问题

**Q: 后端启动失败？**
A: 确保已安装 Java 11+、Maven 3.6+ 和 MySQL 5.7

**Q: 数据库连接失败？**
A: 检查 MySQL 是否启动，数据库 wardrobe 是否已创建，用户名密码是否正确

**Q: 前端启动失败？**
A: 确保已安装 Node.js 16+，运行 `npm install` 安装依赖

**Q: 无法连接后端？**
A: 检查后端是否在 http://localhost:8080 运行

**Q: 忘记密码？**
A: 使用 MySQL 命令行直接修改数据库：
```sql
USE wardrobe;
UPDATE users SET password = '$2a$10$...' WHERE username = 'your_username';
```
