# 智衣橱 - 快速开始指南

## 🚀 运行系统（5分钟搞定）

### 第一步：启动后端

打开终端，执行：

```bash
cd backend
mvn spring-boot:run
```

✅ 看到 "Started WardrobeApplication" 表示后端启动成功！
- 后端地址：http://localhost:8080
- H2控制台：http://localhost:8080/api/h2-console

### 第二步：启动前端

打开新的终端，执行：

```bash
cd frontend
npm install
npm run dev
```

✅ 看到 "Local: http://localhost:5173" 表示前端启动成功！

### 第三步：使用系统

打开浏览器访问：**http://localhost:5173**

1. 点击"立即注册"创建账号
2. 登录后即可使用系统

## ❓ 常见问题

**Q: 后端启动失败？**
A: 确保已安装 Java 11+ 和 Maven 3.6+

**Q: 前端启动失败？**
A: 确保已安装 Node.js 16+，运行 `npm install` 安装依赖

**Q: 无法连接后端？**
A: 检查后端是否在 http://localhost:8080 运行

**Q: 忘记密码？**
A: 使用 H2 控制台直接查看或重置数据库
